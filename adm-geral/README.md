# Painel de Administração Geral (adm-geral) - Challenges Quiz

Este é o diretório do painel de **Administração Geral** do projeto Challenges Quiz.

## O que será desenvolvido aqui?
Esta aplicação será a interface de uso exclusivo dos administradores globais do sistema. Nela serão desenvolvidas funcionalidades como:
- Cadastro e gerenciamento de empresas e clientes parceiros.
- Gerenciamento completo de usuários (criação, edição, desativação de contas de gestores e colaboradores).
- Cadastro, edição e moderação de quizzes, categorias, perguntas e respostas.
- Configurações globais da plataforma (regras de pontuação, limites, temas de personalização).
- Auditoria de logs do sistema e relatórios consolidados de uso global da plataforma.

## Tecnologias propostas (a definir/confirmar)
- React, Vue.js ou Angular.
- Framework CSS ou biblioteca de componentes UI (ex: Tailwind CSS, Shadcn/UI, Material UI).
