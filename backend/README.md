# Backend - Challenges Quiz

Este é o diretório do **Backend** do projeto Challenges Quiz.

## O que será desenvolvido aqui?
Este módulo conterá a API e as regras de negócio centrais do sistema, incluindo:
- Conexão e modelagem de banco de dados (ex: Postgres, MongoDB, etc.).
- Rotas e controladores da API REST ou endpoints GraphQL.
- Sistema de autenticação e autorização (JWT/OAuth) para colaboradores, gestores e administradores.
- Lógica dos quizzes, cálculo de pontuações, gerenciamento de rankings e envio de notificações.
- Integrações com serviços externos.

## Tecnologias propostas (a definir/confirmar)
- Node.js (com Express, Fastify ou NestJS) ou Python (FastAPI/Django).
- Banco de dados relacional ou não-relacional.
