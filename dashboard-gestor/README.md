# Dashboard do Gestor (dashboard-gestor) - Challenges Quiz

Este é o diretório do **Dashboard do Gestor** do projeto Challenges Quiz.

## O que será desenvolvido aqui?
Esta aplicação é voltada para gestores de equipes, departamentos ou líderes de recursos humanos. O foco principal é a visualização e acompanhamento de métricas de engajamento e performance dos colaboradores. As funcionalidades principais incluirão:
- Visualização de painéis (dashboards) com gráficos de desempenho das equipes nos desafios/quizzes.
- Relatórios individuais e coletivos de progresso, acertos e engajamento dos colaboradores sob sua gestão.
- Definição e acompanhamento de metas ou desafios específicos para suas equipes.
- Exportação de dados e relatórios de desempenho.

## Tecnologias propostas (a definir/confirmar)
- React, Vue.js ou Next.js.
- Bibliotecas de gráficos (ex: Chart.js, Recharts, Tremor).
